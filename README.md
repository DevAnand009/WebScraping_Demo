# WebScraping_Demo
Webscraping Demo using ScrapeGraphAI

A compact notebook demonstrating how to use ScrapeGraphAI for guided web scraping and structured data extraction. It will walk you through configuring the ScrapeGraphAI client, building scraping tasks, running crawls on example pages, and saving cleaned, tabular outputs (CSV/JSON) for downstream analysis. Ideal as a hands-on reference for quickly prototyping scraper workflows and extracting structured data from web pages.
